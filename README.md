# 🎮 B站 iOS 充值最优方案计算器

> 一个超可爱、超好用的工具，帮你用最少的点击次数完成B站App Store充值！

你是不是也在为B站iOS充值时要反复点来点去而烦恼？  
这个工具就是来拯救你的！输入你想充的B币金额，它会立刻告诉你如何用最少的操作次数完成充值，省时省力，还能顺便装个“数学小天才”✨

---

## 🚀 功能亮点

- 🎯 **智能计算**：基于动态规划算法，找出最少点击次数的充值方案
- 🍎 **iOS专供**：完美适配App Store固定面额（648、388、318...等）
- 🎨 **B站风格UI**：粉蓝配色，圆润卡片，视觉友好
- 📱 **响应式设计**：手机、平板、电脑都能舒服使用
- ⚡ **实时计算**：输入金额后立即出结果，支持键盘回车
- 📊 **清晰展示**：每一步该点哪个面额、点几次，一目了然

---

## 🛠️ 使用方式

1. 打开页面：<https://b.hicatalpa.cn/>
2. 在输入框输入你想充值的金额（单位：B币对应人民币）
3. 点击 **“计算最省力方案”** 按钮
4. 查看下方生成的“最优点击组合”，照着点就行啦！

> 💡 小提示：你也可以直接按键盘上的 `Enter` 键快速计算哦！

## 🧠 算法说明

工具采用 **动态规划（Dynamic Programming）** 来求解“硬币找零”问题，目标是：
- 在给定的固定面额列表中
- 找出**总次数最少**的组合
- 使总金额等于目标金额

还加入了“大额预减”优化，处理高额充值更高效哦～
